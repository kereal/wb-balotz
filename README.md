# WB Scraper v2.0

Отслеживание новых товаров с акцией **"Баллы за отзыв"** на Wildberries.

## 📦 Установка

```bash
unzip wb-scraper.zip
cd wb-scraper
npm install
```

## 🚀 Использование

### Один раз запустить

```bash
npm run scrape
```

### Запустить веб-интерфейс

```bash
npm run web
# Открыть http://localhost:3001
```

### Автоматический запуск по расписанию

```bash
# Каждый час (по умолчанию)
npm run watch

# Каждые 30 минут
INTERVAL_MINUTES=30 npm run watch
```

## 📋 Команды

| Команда | Описание |
|---------|----------|
| `npm run scrape` | Запустить скрапер |
| `npm run web` | Запустить веб-интерфейс |
| `npm run stats` | Показать статистику |
| `npm run new` | Показать новые товары |
| `npm run history` | Показать историю сканов |
| `npm run watch` | Автозапуск по расписанию |

## 📁 Структура

```
wb-scraper/
├── scraper.js       # Основной скрипт
├── server.js        # Веб-сервер
├── watcher.js       # Автозапуск по расписанию
├── logger.js        # Система логирования
├── products.db      # База данных SQLite
├── public/          # Веб-интерфейс
├── logs/            # Логи (создаются автоматически)
└── data/            # Экспорт JSON
```

## 📊 База данных

### Таблицы

- `all_products` — все найденные товары
- `new_products` — только новые товары
- `scrape_history` — история запусков

### Поля товара

| Поле | Описание |
|------|----------|
| `wb_id` | ID товара на WB |
| `name` | Название |
| `price` | Цена до скидки |
| `sale_price` | Цена со скидкой |
| `brand` | Бренд |
| `seller` | Продавец |
| `review_rating` | Рейтинг |
| `feedbacks` | Кол-во отзывов |
| `url` | Ссылка |
| `first_seen` | Когда впервые найден |
| `last_seen` | Когда последний раз виден |

## 🌐 API

| Endpoint | Описание |
|----------|----------|
| `GET /api/new` | Новые товары |
| `GET /api/all` | Все товары |
| `GET /api/stats` | Статистика |
| `GET /api/history` | История сканов |
| `GET /api/search?q=` | Поиск |
| `DELETE /api/new` | Очистить новые |

## ⚙️ Настройки

Измените в `scraper.js`:

```javascript
const CONFIG = {
  maxRetries: 5,        // Макс. попыток
  baseDelay: 5000,      // Базовая задержка (мс)
  maxDelay: 60000,      // Макс. задержка (мс)
  waitAfterLoad: 20000, // Ожидание после загрузки
  scrollCount: 10,      // Кол-во прокруток
};
```

## 📝 Логи

Логи сохраняются в `logs/scraper-YYYY-MM-DD.log`

Старые логи (>30 дней) удаляются автоматически.

## 🔧 Как это работает

1. Открывает страницу акции в браузере
2. Перехватывает API ответы с товарами
3. Сравнивает с базой — находит новые
4. Сохраняет в SQLite и JSON

## ⚠️ Важно

- Wildberries защищается от ботов
- При ошибках скрипт автоматически повторяет попытки
- Чем чаще запускаете — тем точнее отслеживание
